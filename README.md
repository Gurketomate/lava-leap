# Lava Leap

Lava Leap is a fast-paced platform jumping game where players escape rising lava while collecting coins and using special power-ups.

## Features

- **Level Mode** with multiple handcrafted levels
- **Endless Mode** focused on reaching maximum height
- **Multiple platform types** — lava, ice, moving, coin, crack, and rare shield platforms
- **Power-up items** — magnet, double jump, lava slow, shield
- **Coin system and upgrades** — spend coins on permanent upgrades in the shop

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- HTML5 Canvas (game rendering)

## Getting Started

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
